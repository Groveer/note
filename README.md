# linux-note

#### 介绍
linux-note是用于记录使用Linux操作系统的过程中遇到的一些问题与解决办法，以后遇到同样的问题可以在此来进行查找，由于是项目初期，暂时不予分类，等内容丰富后再进行分类管理。

#### 项目架构
项目采用markdown语法进行展示，如想进行贡献，请使用md文件格式，文件名要简单表达出文件的内容或含义。

#### 使用说明
由于采用markdown编写，可以直接在线查看。
